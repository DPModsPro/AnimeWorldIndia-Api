AniBiee API

A complete RESTful solution for AnimeWorldIndia data. Fast, reliable, and developer-friendly.

Documentation Repo
Discovery Endpoints
GET/api/home

Get trending, newest drops, and movies list for the landing page.

GET/api/category/{type}?page={n}

Browse categories (movies, series, genre/action, language/hindi).

Example: /api/category/movies?page=1
Content Details
GET/api/info/{id}

Get detailed info, description, and episodes list of a specific title.

Example: /api/info/spy-x-family
GET/api/episodes/{id}/{season}

Fetch all episodes for a specific season of a series.

Streaming & Tools
GET/api/embed/{episodeId}

Get player server links (Play, Abyss, etc.) for streaming.

GET/api/scrape?url={targetUrl}

Generic scraper to extract data from any supported URL.

GET/api/health

Check server uptime and API status.
